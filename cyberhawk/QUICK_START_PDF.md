# 🚀 Quick Reference - PDF Report Download

## Dashboard Features Overview

```
┌─────────────────────────────────────────────────────────────┐
│  🛡️ CyberHawk                                     📥 Download │
│  Cyber Threat Intelligence System                  PDF Report │
└─────────────────────────────────────────────────────────────┘
```

## Three Easy Steps

### 1️⃣ Input Network Features
```
Enter comma-separated values (9 features required):
100, 5000, 10, 6, 0, 1023, 80, 10, 500
```

### 2️⃣ Click "🔍 Detect Threat"
The system analyzes the network traffic and displays:
- Threat Type (e.g., Normal Traffic, DoS Attack, Malware)
- Risk Level (Low, Medium, High, Critical)
- Status Indicator (Safe ✅ or Detected 🚨)

### 3️⃣ Click "📝 Generate & Download PDF"
Your professional PDF report is ready instantly!

## Download Locations

### **Header Download Button** (Top Right)
- Available after report generation
- One-click access
- Always visible when report exists

### **Result Section Download Button**
- Below the threat analysis details
- Direct download from analysis results
- Shows full filename

## What's Inside Your PDF Report

